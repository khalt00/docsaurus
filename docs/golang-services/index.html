Run service to receive event from smartcontract

  • cd to this project https://github.com/khalt00/Timekeeping-go
  • create a config.yaml file for docker compose use
    engine: "postgres"
    host: "db"
    port: "5432"
    database: "attendance"
    username: "postgres"
    password: "postgres"
    sslmode: "disable"

    alchemy_url: "" #Polygon mumbai wws url
    smart_contract_address: "" # deployed smart contract address

  • Run this command docker compose up -d

  • Wait for it to run success. You can use this command to inspect it when it started docker compose logs -f

  • Please connect to some posgres client. Suggest using pgadmin on window or dbeaver on linux

  • Go to our hardhat project, copy all abi

  • Go to Remix IDE, create a new myabi.abi in contracts folder
  • Paste in

  • While still select that abi file, go to Deploy & run transactions
  • Copy your smartcontract address to At address
  • It will look something like this

  • Add first record to our record
  • Note: You must be the owner of the contract, or else the owner have to add you to User so you can record attendance

  • After finish checkedin, check your backend and database.
  • Backend will have log like this

  • Database will inserted the event
